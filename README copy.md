## مين أنا؟ 👋

بصراحة، مش بحب الـ bios اللي بتقول "Passionate developer with X years of experience" — دي template مش bio.

دلوقتي بشتغل على مشاريع تجمع بين:
- **Backend** سريع وموثوق مع **Go** و**Laravel**

- **Mobile** cross-platform مع **Flutter**

مش بتخصص في حاجة واحدة عشان الدنيا محتاجة ناس تفهم الصورة كاملة.

## تجارب شكّلتني 💭

**تجربة #1 — لما اخترت Go**

كنت بشتغل على API بـ Laravel وكان كل حاجة تمام — لحد ما الـ load زاد. وقتها قررت أجرب Go، وكان الفرق في الـ performance صادم. مش معناها إن Laravel مش كويس — معناها إن الـ tool الصح في الوقت الصح بيفرق.

**تجربة #2 — Full Stack مش "كل حاجة"**

ناس كتير بتفهم Full Stack غلط — بتفتكر إنه يعني "بعمل كل حاجة بشكل متوسط". أنا بفكر فيه بشكل تاني: إنك تفهم الـ system كله عشان تاخد قرارات أحسن في الجزء اللي بتشتغل فيه.
